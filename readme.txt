运行本项目前请先查看application.properties文件中的各种配置是否符合您的运行环境：
1 请根据实际情况来配置连接池（JDBCURL，用户名，密码等）；
2 该项目涉及到邮件发送，请自行完成邮件模块的相关配置（邮箱及密码等）；
3 属性my-config.upload-photo-dir表示的是用户上传头像时图片的物理存储路径，请根据实际情况来填写。
